# 🚀 Stellar Campus Rewards

🔗 Live Project: https://github.com/zeydaneng/stellar-campus-rewards
Empowering students with blockchain-based rewards on the Stellar network.

---

## 📌 Project Description

Stellar Campus Rewards is a decentralized application built on the Stellar Testnet that allows universities and organizations to reward students with XLM tokens for completing academic and extracurricular activities.

Students can submit tasks such as workshops, volunteering, or projects. Administrators review and approve these tasks, then send rewards directly to the student's Stellar wallet using secure blockchain transactions.

This ensures transparency, fairness, and trust in the reward distribution process.

---

## 🌍 Vision

Our vision is to create a global, transparent, and decentralized student reward ecosystem powered by blockchain technology.

By leveraging Stellar, we aim to eliminate bias, delays, and inefficiencies in traditional reward systems.

---

## ⚙️ Features

- 🔗 Connect Freighter Wallet
- 🧑‍🎓 Submit student tasks
- ✅ Admin approval system
- 💰 Send XLM rewards
- 📊 Track funded tasks
- 🌐 Transparent blockchain transactions

---

## 🛠 Tech Stack

- Frontend: React + Vite
- Blockchain: Stellar (Soroban Testnet)
- Wallet: Freighter
- Smart Contracts: Rust (Soroban)

---

## 🚀 How to Run the Project

```bash
git clone https://github.com/zeydaneng/stellar-campus-rewards.git
cd stellar-campus-rewards
cd frontend
npm install
npm run dev
```

Open in browser:

```
http://localhost:3000
```

---

## 🔌 Wallet Setup

1. Install Freighter Wallet
2. Switch to Testnet
3. Fund your wallet using Stellar Laboratory
4. Connect wallet in the app

---

## 💸 Demo Flow

1. Connect wallet
2. Submit a task

4. Approve task
